# MapLocation
Utilizzo delle API di Google su Mappa Device

- Dipendenze

installare CocoaPod


$sudo gem install cocoapods

- CocoaPod

pod 'GoogleMaps', '~> 2.1'

pod 'GooglePlaces', '~> 2.1'


- Google Key

Cambiare la chiave di Google nel file AppDelegate

- Avvio in Xcode

Avviare il file-> Location.xcworkspace
